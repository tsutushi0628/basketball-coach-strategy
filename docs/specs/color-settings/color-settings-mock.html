チームカラー設定 — 設計モック

各チームが自分のメインカラーを厳選16プリセットから選ぶ設定面。右上のログアウト隣に歯車を置き、押すとプリセット選択パネルが開く。色・書体・罫線はすべて練習計画本体と同じトークン。クリーム地・男女別の地色は据え置き、ブランド主色だけ差し替わる。

このページは設計確認用。各ステージ枠は本番ページの右上ヘッダ領域を相対配置で再現したもの（本番は position:fixed）。歯車・パネルの状態と、16テーマを実際に当てたスウォッチの見た目を1枚で確認できる。明色テーマ（スカイ・ライム・ゴールド）は主色面の文字を暗インクにしている。
