# オフェンス概説（ポストトリガー）

> 出典: （Googleドキュメント）「ポストトリガーのオフェンス概説」

# 目的

- ハーフコートオフェンスの選択肢を増やす

# 現状

- ハンドラーアタックがオフェンスパターンの多くを占めるが、イージーバスケットが少ない（タフショットが多くなりがち）
- 主立ったハンドラーの選手がタイトにディフェンスされていると、ハーフコートオフェンスが出来ない

# 課題の解決策

- ポストマン（ショートコーナーにポジショニングしている選手）をトリガーとした、ハンドラーとのコンビネーションオフェンスを出来るようにする

# 課題の解決法

- ポストマンにボールを…

    - 入れない場合、ポストマンはハンドラーに対してスクリーンをかける（以下ポストマンをピックマン、ハンドラーのマークをハンドラーD、ピックマンのマークをピックマンDと呼称する）

        - ハンドラーの動きは以下

            - リムアタック

                - がっちりスクリーンかかったら

            - スプリット

                - スイッチ（ヘッヂ）への対抗策。ハンドラーDとピックマンDの間を抜く

            - スネーク

                - ドロップ（アンダー）への対抗策。コースを曲げる

            - リジェクト

                - スクリーンをデコイにする

            - プルアップスリー

                - ドロップ（アンダー）への対抗策

        - ピックマンの動きは以下

            - ダイブ（ロール）

                - ゴールに飛び込む

            - スリップ

                - スクリーンと見せかけてゴールに向かう

            - アングルピック（リピック）

                - オーバーやアンダーでハンドラーDにかわされた場合、ハンドラーの動きに合わせて再度スクリーンを試みる

            - ポップ

                - アウトサイドに出る

    - 入れる場合、ポストマンはショートコーナーからハイポストに向けてフラッシュしボールをレシーブする

        - そこからの攻め手は主に以下の4パターン

            - ターン＆シュート
            - リムアタック

                - 4アウト1インのシチュエーションなら勝ち目はある

            - ハンドオフ

                - カットインしてきたハンドラーに手渡し
                - 自身はスクリーナーも兼ねる

            - コーナーからのカッティングにパスを合わせる

        - ハンドラーは、ポストマンの挙動に合わせて次の動きを決める

            - ゴールを向かなければパス＆ラン
            - ゴールを向いたならボールサイドのコーナーにダウンスクリーン

                - コーナーの選手がカールカット気味からのリムアタックを狙う

        - ポストマンにボールが入らなかった場合、UCLAカットにつなげる

# ポイント

## 共通

- ハンドラーはスリーポイントラインから2歩ほど離れた位置にいること。これは、ポストへのパスにせよスクリーンにせよ十分なスペースを空ける必要があるからである
- スクリーンかポストアップか（ドライブでのハンドラーアタックか）の判断は、ハンドラーがするのが望ましい

## スクリーン編

- 目的は強制的に2on1のシチュエーションを作ることである

### ハンドラー編

- ハンドラーは、ハンドラーDのスクリーンに対する動きでどの挙動を選択するか決める
- スクリーンを使うときはピックマンのギリギリを通る
- ダイブへのパスを通すときは、「ワンハンドかつバウンズ」をまず狙う

### ピックマン編

- イリーガルスクリーンに注意。ハンドラーDの進路を故意に妨害するような挙動を見せない

    - とはいえ、動きの中では積極的にハンドラーDをクリアアウトするようにする

- ハンドラーの進路と身体の正面を垂直にする
- ダイブ時、ハンドラーとのスペースに注意。可能な限り広く取る

## ポストムーブ編

- 目的は相手に対してインで攻めるかアウトで攻めるかを絞らせないことである

### ハンドラー編

- パスを出すタイミングは、ポストマンがフラッシュし終えたその瞬間である。遅くても早くてもいけない
- 基本はパス＆ラン

### ポストマン編

- 動きには緩急をつける。特にフラッシュ時、だらーっと上がらない
- フラッシュとは言え、マークマンをジェイル気味にシールする
- ターゲットハンドを忘れない

    - ハンドラーがパスを出しやすくなるとともに、ネクストアクションの候補を伝えることとなる
