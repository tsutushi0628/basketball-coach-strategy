## 試合の基本構成

中学生のバスケットボールは、公式戦の場合 1クォーター（Q）8分を4回行います。

| 区間 | 時間 |
|------|------|
| 1Q | 8分 |
| 2Q | 8分 |
| ハーフタイム（2Q〜3Q間） | 10分 |
| 3Q | 8分 |
| 4Q | 8分 |

1Qと2Q、3Qと4Qの間には2分間の休憩、2Qと3Qとの間には10分間の休憩（ハーフタイム）が設けられています。

---

## 時計が止まるケース

以下の場合は毎回時計が止まります。

- コートの外にボールが出た場合
- ファウルがあった場合
- フリースロー
- 監督が選手に指示を出す時間「タイムアウト」など
