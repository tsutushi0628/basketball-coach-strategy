# チーム基本情報（自校バスケットボール部）

令和8年度の保護者会資料（令和8年5月8日）をもとにした、部の方針・体制・年間スケジュール・運営の基礎情報。実名（顧問・外部指導員・選手）と学校の連絡先は、本ナレッジベースのPII方針によりマスクしている（個々の選手は計測台帳の選手名簿シートで管理）。

## 体制

- 顧問: 2名
- 外部指導員: 2名

## 部の方針（活動方針）

- 集団行動の中で、自己を発見し、仲間との協力ができる人間性を養う。
- あいさつができる、けじめをつける、仲間を思いやる。
- 将来のために、社会のルール・マナーを身に付ける。
- バスケを通して、スポーツの厳しさと楽しさを学ぶ。
- バスケの知識・技能を高める。

（バスケットボールの戦術的なDF/OFフィロソフィーは [../tactics/チーム哲学とセットオフェンス.md](../tactics/チーム哲学とセットオフェンス.md) を参照）

## 運営（保護者向け・参考）

- 部費: 5,000円（JBAのチーム登録料・個人登録料を含む）
- 準備物: バスケットボールシューズ、ユニフォーム（女子は学校貸出／男子は2年生から新デザインを購入）、リバーシブルノースリーブ（ビブス）、練習用T、練習用長袖T、チームトレーナー、ウインドブレーカー。練習着は黒・紺無地の短パン＋ワンポイント程度の白T可。
- 欠席連絡: Google Classroom へ連絡。
- 怪我: 子ども医療証は使わず、スポーツ振興センターでの支払い。
- 大会時の保護者向け注意: 男女問わず観戦可／会場への車・バイク乗り入れ不可／会場で保護者の食事不可／差し入れはアレルギー事故防止で全部活なし／他校試合の観戦は原則禁止（キリンレモンスポーツセンター会場を除く）／フラッシュ撮影禁止／室内履き持参／審判への判定クレーム不可。
