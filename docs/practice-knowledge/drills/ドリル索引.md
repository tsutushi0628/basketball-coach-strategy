# ドリル索引（能力・指標タグ表）

この索引は「伸ばしたい能力・動かしたい数字」から逆引きで練習種目を選ぶための表である。コーチが練習計画を立てるとき、まず「今どの能力／どの指標を上げたいか」を決め、この表でその能力に効くドリルを探し、出典を辿って手順を確認する、という使い方を想定している。各ドリルの具体的な手順・約束事・バリエーション・参考動画はこの表には複製していない。手順を引くときは出典欄に示した [../drills/ドリルカタログ.md](../drills/ドリルカタログ.md)（全ドリルの源）と [../tactics/](../tactics/)（セットプレーの動き方）を参照すること。「いつ・どの順で・何分やるか」の練習計画は [../sessions/](../sessions/) を見る。

表の読み方:

- **動く指標**: そのドリルを継続すると改善が期待できる、計測可能な数字。スタッツの語彙は [../reference/スタッツとシュート分析.md](../reference/スタッツとシュート分析.md) に準拠（FG%／3FG%／FT%／エリア別成功率／TO／OR／DR／EFF など）。数字に落とし込めないものは「指標化候補」と書き、末尾の[指標ギャップ表](#指標ギャップ表)で測定方法案を示す。
- **必要環境**: コート面（全面／半面／¼）、ボール数、人数、大人の手伝い要否を短縮表記。
- **対象レベル・学年**: 学年は 1〜3／全 などで表記。中学生は成長期スパート期にあたるため、ジャンプ・全力ダッシュ・連続着地を含むドリルには負荷注意を併記する。
- **習熟段階**: 習得（新しい動きを覚える）／反復（フォームを固める）／実戦化（プレッシャー下・ゲーム文脈で使えるようにする）の3段階。

## 指標ギャップ表

上の各表で「指標化候補」と記したドリルについて、何を測れば成長を数字で追えるかの測定方法案をまとめる。既存のスタッツ分析シート（FG%・エリア別成功率など）はゲーム/シュート結果の事後計測に強い一方、ウォームアップ・ハンドリング・守備姿勢などの「過程能力」を測る仕組みが薄い。ここを埋める提案。

| ドリル名（指標化候補） | 何を測るか | 測定方法の案 | 記録の置き場（案） |
|---|---|---|---|
| ダイナミックストレッチ | 怪我の予防効果 | 月次の故障・離脱件数を記録し推移を見る | コーチノートに月次集計 |
| ラダートレーニング | 足さばきの速さ・正確性 | 規定メニュー1往復の所要秒数＋ミス回数を月1回計測 | 個人カードに月次タイム |
| 5ボディビルダーズ / HIIT4分間 | 筋持久力・無酸素持久力 | 規定時間内の反復回数（例: HIIT各20秒の回数）を記録 | 個人カードに回数 |
| フットワーク／階段ダッシュ／シャトルラン | 加速・持久 | 一定区間/本数のタイムを月1回計測（5-10-5アジリティ等を追加） | 個人カードに月次タイム |
| サークルドリブル／ハンドリングドリル | ハンドリング安定 | 30秒間の落球回数、または規定本数のミス無し継続を計測 | 個人カードに回数 |
| 片足バランス＆チェストパス | 体幹安定 | 崩れずに連続できたパス本数 | 個人カードに回数 |

## 不足ドリル候補

中学生男女のチームに対し、既存カタログに無い／薄く、補うと効果が見込める種目を理由つきで挙げる。スタッツ（[../reference/スタッツとシュート分析.md](../reference/スタッツとシュート分析.md)）に表れた弱点（FT%が全体に低い・TOが多い・チーム3FG%が低い・DFリバウンド課題・トランジション戻りの課題）と紐づけている。

- **プレッシャー下フリースロー（疲労＋連続でのFT）**: 試合のFT%が軒並み低く（多くの選手で50%未満、フリースロー指摘がコーチコメントに頻出）、静かな環境での反復だけでは試合状況に転移しない。ダッシュ後・ノイズ下・連続2本などの負荷をかけたFTドリルが薄い。
- **2ボール／ライブ・ハンドリング（DF付き）**: TOが選手横断で多い。既存ハンドリングは静的中心で、DFプレッシャー下で顔を上げてキープする実戦的ハンドリングが薄い。
- **キャッチ＆シュートのライブ・クローズアウト付き反復**: チーム3FG%が低く（主要シューター以外ほぼ0）、スタティック/ダイナミックの個人練はあるが「DFがクローズアウトしてくる中で構えて打つ」反復が不足。3FG%の底上げに直結。
- **DFリバウンド（ボックスアウト）の対人徹底ドリル**: コーチメモで「DFリバウンド」「視線は外回り」「下3人でトライアングル」が繰り返し課題化。ボックスアウト個別ドリルはあるが、5対5での確実な確保を反復する専用ドリルが薄い。
- **トランジションDF（戻り）ドリル**: 「まずフリースローレーンに向かって戻る」が複数試合でメモ化。攻→守の切替で素早く帰陣し陣形を作る専用ドリル（例: 3on2→2on1コンバージョン、ジャマー付きの帰陣競争）が無い。
- **クロースアウト→1on1のクローズアウト守備ドリル単体**: スリーコーンリアクションに含まれるが、被3P/被ドライブの分岐を判断する単独反復として切り出すと守備個人技が上がる。
- **ピボット／トラベリング矯正の専用ドリル**: コーチコメントでトラベリングが明示課題。ストップ→ピボット練はあるが、ゼロステップ/ギャザー理解を反復で体に入れる専用枠が薄い。TO低減に直結。
- **コミュニケーション／コール練習（声出し守備）**: ヘルプ・スクリーンコール等の声の連係を測る/育てる枠が無い。シェル/3on3に組み込む形で「コール必須ルール」を加えると守備連係が上がる。
- **女子向けの上半身パワー・ボディコンタクト馴致**: 男女混在チーム想定なら、接触に対する体の入れ方・押し負けない体幹を安全に養うコンタクトドリル（マット併用等）を成長期負荷に配慮して追加する余地。
- **持久系の客観計測ドリル（ヨーヨー/ビープテスト相当）**: コンディショニングは実施しているが「数字で持久力を追う」測定種目が無く、成長の可視化ができない。指標ギャップ表とセットで導入すると効果的。
