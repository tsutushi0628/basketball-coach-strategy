# 基本の練習（外→多目的室）（平日55分）

## ダイナミックストレッチ（簡易版、外でも出来る）

- 走る日向け。下半身・足首・ハム優先で、上半身と高衝撃ジャンプは省略（硬い地面＋走る日は接地負荷を避ける）
- 膝とつま先は必ず同じ向きにする

1. パワーウォーク＆股関節ストレッチ＆足の裏の筋伸ばし（温め＋股関節＋ふくらはぎ）
2. ランジウォーク（後半は身体をひねる）
3. ヒップローテーション
4. キャリオカステップ
5. ポゴジャンプ（足首準備・低衝撃）
6. フロントキック（動的ハム）→ そのまま校庭ランの最初の数周をビルドアップにして繋ぐ

## 校庭ラン

- 普通に5周
- 残り5周は5秒全力ダッシュ→10秒歩き
