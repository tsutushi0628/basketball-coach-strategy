# 基本の練習（体育館）（平日100分、土曜日165分）

## アップ（20分）

### ダイナミックストレッチ
- RAMP（Raise温める→Activate効かせる→Mobilize動かす→Potentiate爆発）
- 膝とつま先は必ず同じ向きにする

1. パワーウォーク＆股関節ストレッチ＆足の裏の筋伸ばし
  - 半分までパワーウォークで進み、その後股割り
  - 後半はつま先を触りながら後ろに進む
2. ランジウォーク
  - 大きく踏み込み、
  - 前半：手を上げて肩を伸ばす
  - 後半：身体をひねる
3. 肩甲骨スキップ
  - 腕を肩から上に上げる、スキップしながら
  - 後半はバックスキップ
4. 肩回し
  - サイドステップしながら肩を回す
  - 後半は身体の向きを変える
5. ヒップローテーション
  - 膝を腰の位置まで上げて外側から内側に向けて回す
  - 後半は内側から外側、後ろ向き
6. キャリオカステップ
  - 脚を前後に踏み換えて腰をツイストしながら進む
  - 後半は身体の向きを変える
7. スキップ（エルボー to ニー）
  - 手は頭の後ろで組む。手と足をクロスし、膝を肘にくっつけながらスキップ
  - 後半はバックスキップ
8. ポゴジャンプ
  - 手を上に組み、足首だけでジャンプ。後半はバック
9. フロントキック
  - 脚を上げて、手の先とつま先を触れさせる
10. バウンディング
  - パワーポジションのまま両足で床を蹴って後ろ向きに進む
  - 前半大きいキック、後半小さいキック
11. ジャンプキック
  - 可能な限り高く
12. ブロードジャンプ＆スティック着地
  - 立ち幅跳び。可能な限り大きく
13. ハーキー→ダッシュ
14. もも上げ→ダッシュ
15. サイドステップ→ダッシュ

### ダイナミックストレッチ 省略版（1時間枠のとき・約10分）
- ウォームアップは0にしない（怪我＋パワー低下）。温め＋股関節・ハム・足首＋数本ビルドアップだけ残す
- 最初に切る：肩甲骨スキップ・肩回し（上半身）、バウンディング・ジャンプキック・ブロード＆スティック（大ジャンプ）
- 最初のドリルを低強度にして"続きの温め"にすると、正味5分でも体感は十分上がる

1. パワーウォーク＆股関節ストレッチ＆足の裏の筋伸ばし（温め＋股関節＋ふくらはぎ）
2. ランジウォーク（後半は身体をひねる）
3. ポゴジャンプ（足首）
4. フロントキック（動的ハム）
5. ハーキー→ダッシュ ×1〜2本（試合速度へビルドアップ）

    - フロントターン
