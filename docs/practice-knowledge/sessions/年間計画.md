# 年間計画（ピーキング設計）

令和8年度、新チーム（夏の代替わりで発足する1〜2年）の1年を設計する骨組み。[チーム基本情報](../reference/チーム基本情報.md) の年間大会カレンダーとシーズン構造に基づく。発足のきっかけになる夏の大会（中野区選手権〜都選手権）は「前の代」の3年の集大成で本計画の対象外。ここで代替わりして新チームが発足する＝新チームにとって夏は出発点。新チーム自身の山は2つ——冬の新人大会（第1の山）と、翌夏の中体連（この代が3年として迎える集大成・最大の山）。 日々の具体メニューは [練習計画.md](練習計画.md)（担当別の曜日設計・KGI/KPI・基本形）と [../drills/ドリル索引.md](../drills/ドリル索引.md) を組み合わせて組み、伸びは [計測台帳](../reference/計測台帳マップ.md) で追う。

## 設計の前提

- 男子・女子は別計画。体育館は共有（火・金＝全面で一部共通/一部分割、水・木＝半面でほぼ共有）。割り当ては変更可なので、週次プランで最適化する。
- 目標は「男女とも、この代の最後の大会＝翌夏の中体連で2勝」（コーチ確定）。新人大会（冬）は集大成への通過点の山。「2勝」は直接は練習できないので、勝つために必要な process 指標（下記）に翻訳して計測台帳で追う（指標の目標値は後日設定）。
- 中学生は成長期。高負荷ジャンプ・全力ダッシュは頻度を管理し、基礎づくりの時期に土台を作る。
- フィロソフィー（[チーム哲学](../tactics/チーム哲学とセットオフェンス.md)）: DF＝常にオールコートマンツー一本（中学ゾーン禁止）／OF＝アーリーオフェンス基調・ハーフコートでも15秒以内にシュート。セットプレーは速さができた上で後から足す（順序は不可逆）。

## 新チームの年間サイクル（夏に発足 → 冬の新人大会 → 翌夏の中体連で集大成）

新チームにとって夏は「山」ではなく出発点。山は2つ——冬の新人大会と、翌夏の中体連（この代の集大成）。

### 出発点: 代替わり（7〜8月）

- 夏の大会（中野区選手権〜都選手権）で現3年が引退 → 1〜2年の新チームが発足。これが新チームの起点。
- 男子: 都選手権まで進む見込み（7月末）＝発足はやや後ろ倒し。女子: 早めに発足。
- 夏休み（8/6〜8/16）はオフ。男女で始動時期がずれる。
- ※夏の大会そのもの（現3年の集大成）は本計画の対象外。新チーム発足のトリガーとして置くだけ。

### 第1の山: 新人大会（冬）

- 中野区新人大会（11〜12月）＝新チーム最初の山。ここにピーキングを合わせる。
- 東京都新人大会（1〜2月）＝勝ち上がれば山が2月まで伸びる。

### 第2の山＝集大成: 中体連（翌6〜7月）

- 3月の研修大会・ムサキタカップのあと、翌4月に学年が上がり新1年が加入。4〜5月で仕上げ、6〜7月の中野区選手権→（勝ち上がれば）都選手権が、この代（いまの2年が3年）の集大成・最大の山。
- ここを終えると3年が引退し、次の新チームへ。サイクルが一周する。

### 準備期（8〜9月）にやること — 得点を軸に

新チームの土台は「点の取り方」を中心に置く。現代バスケでは孤立した基礎練習だけでは試合に転移しないため、以下を別々のブロックにせず実戦の中に溶かし込んで身につける。比重は個〜小集団の得点の基礎だが、人数は縛らない——DFの基礎は3on3が要るし、ゲーム感覚として5on5も使う。

- ハンドリングのファンダメンタル
- シュート率向上（近〜中距離。3P・FTは鍛錬期以降に層を足す）
- ディフェンスの基礎の約束事（1線・2線・3線、ヘルプ、ディナイ、クローズアウト）。1線2線3線は3on3でないと学べないので3on3を使う
- ボックスアウトのファンダメンタル
- 対人で点を取る（1on1・2on2が中心）。ゲーム感覚として5on5も入れる

準備と鍛錬の違いは「人数」ではなく「攻め方の質」。年内の攻めはアイソレーション（個で点を取る）が上限で、セットプレーや決まったチームシステムは組まない（入れるとしても年明け以降、チームの仕上がり次第）。

### 8月シード権大会で「試せるもの」（最小デプロイ）

立ち上げ直後でも、ハンドリング等の孤立ドリルだけでは試合にならない。点を取る／守るの最小限の「使える型」を持って臨む。年内アイソの方針どおりセットプレーは入れない。

- 速攻・アーリーオフェンス: 走って早く点を取る。準備が最も軽く、新チームの主武器になる。
- ハーフコートOF ＝ スペーシング＋ドライブ&キック: 4アウト/5アウトに広がり、1人が1on1でアタック、他はキャッチ&シュート準備。これは「組まれた型」ではなくアイソの土台なので年内方針に沿う。
- オールコートマンツー（DFはこれ一本）: 全コートで仕掛けるマンツーマン。中学はゾーン禁止なので守りはこれだけ。1線2線3線＋ボール圧＋ヘルプはこの枠の中で仕込む。
- 安全なインバウンド（BLOB/SLOB）: 凝らず、確実に入れる＋簡単な合わせを1つだけ。

DF方針: 常にオールコートマンツー一本（中学はゾーン禁止＝そもそもゾーンは使えない）。できる/できないに関わらずこれで通す。スキームを切り替えない分シンプルで、体力と対人守備がそのまま育つ。シード権大会も新人大会もこのまま臨む。

## 「2勝」を測れる目標に落とす

集大成（翌夏の中体連）で2勝するために——そして冬の新人大会でも力試しで勝つために——勝敗を分ける process 指標を計測台帳で追う。例:

- フリースロー成功率（FT01） … 接戦はFTで決まる
- 試合ターンオーバー（GM_TO） … 自滅を減らす
- ゴール下/レイアップ成功率（UND01・LAY01） … イージーを確実に決める
- トランジションDF … 被速攻を減らす（[ドリル索引](../drills/ドリル索引.md) の不足ドリル候補にも挙げた帰陣ドリルが該当）

具体的な目標値は、計測台帳「目標」シートにベースライン測定後に置く。

## 月次PDCA

- 月初: その月のフェーズの主眼を確認し、週次プランに落とす。
- 月末: 計測ログで指標の伸びを見て Fit&Gap（狙い通り伸びたか）を判定し、翌月の練習比重を調整する。
