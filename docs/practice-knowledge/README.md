# バスケ練習ナレッジベース

コーチ業で蓄積した練習・戦術の知識を、Claude に相談しやすい形でまとめた知識ベース。元データは Google Drive（個人アカウント）の練習関連ドキュメントで、2026-05-30 時点の内容を Markdown 化したもの。

内容の性質ごとに ドリル／タクティクス／セッション／リファレンス の4カテゴリに分けて格納している。

## カテゴリ構成

| フォルダ | 中身 | 何を引くか |
|---|---|---|
| [sessions/](sessions/) | 日々の練習カリキュラム（時間割）・計画・記録 | 「いつ・何分・何をやったか」 |
| [drills/](drills/) | 練習種目そのもののリスト | 「どんなドリルがあるか」 |
| [tactics/](tactics/) | 戦術・セットプレー・チーム哲学 | 「どう攻める／守るか（X's and O's）」 |
| [reference/](reference/) | ルール・スタッツ・振り返りフォーマット・その他 | 「基礎情報・データ」 |

## 使い方（Claude への相談例）

- 「PnR の指導、どう組み立ててた？」 → [tactics/PnR講座.md](tactics/PnR講座.md) ＋ [tactics/試合戦術.md](tactics/試合戦術.md)
- 「うちのディフェンスの基本方針は？」 → [tactics/チーム哲学とセットオフェンス.md](tactics/チーム哲学とセットオフェンス.md) ＋ [sessions/練習メニュー集_2022-2023.md](sessions/練習メニュー集_2022-2023.md) 末尾「チーム方針の変遷」

## 元データ（Google Drive）との対応

- 練習関連フォルダ2つ ＋ サブフォルダ「男バス/練習」等を走査。
- 図解中心のPDF（自主練・ルール）と大容量プレゼン（25〜60MB）は、テキスト変換で取れる範囲のみ転記し、図解は「原本参照」と明記（取れなかった部分を取れたように見せていない）。
- 戦術プレゼンのプレー図（コート図）の画像取り込みによる文章化は、外部ツールの上限により今回未実施（後日追補予定）。

## このナレッジベースに含めていないもの（理由つき）

- 選手個別フィードバック（個人別ノート）／卒業生へのメッセージ — 個人評価・私信のため知識ベースから除外
- 写真・動画（mp4） — テキスト知識ではないため
- 予定表（スケジュール表） — 日時の運用情報で練習知識ではないため

## 個人情報の取り扱い

選手・スタッフ・学校・対戦相手の実名は、知識としての価値を保ったまま伏字（選手A〜H／選手1〜11／コーチ／相手校／自校 など、同一人物は同一トークンに固定）に置換済み。背番号・スタッツ等の数値はそのまま保持している。
