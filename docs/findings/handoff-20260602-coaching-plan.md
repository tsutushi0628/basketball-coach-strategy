## 議論の主要トピックと到達点（コーチング哲学の確定事項）

新セッションはこの章を読めば設計判断の理由を即座に把握できる。出典は tactics/チーム哲学とセットオフェンス.md と sessions/年間計画.md。

- DFフィロソフィー: 常にオールコートマンツー一本。中学はゾーン禁止（ルール）。できる/できないに関わらず通す＝スキームを切り替えない分シンプルで、体力と対人守備が育つ。
- OFフィロソフィー: アーリーオフェンス基調。速攻はドリブルプッシュ（リバウンドを取った選手がサイズに関係なく自分で真ん中を運ぶ＝ポジションレス。長い静的アウトレットはしない。例外は囲まれた時だけ味方がカール気味に短く受ける）。ヒットアヘッドは確実に安全なときだけ（セーフティがいるのが普通なので相手を引きつけてから判断）。ハーフコートでも15秒以内にシュート。セットプレーは速さを土台にした後から足す（順序は不可逆＝遅いパス回しを先に刷り込むと速い攻めの本能が育たない）。
- 年間構造: 新チーム（夏の代替わりで発足する1〜2年）の約1年を設計。発足トリガーの夏の大会（前の代の3年の集大成）は対象外。山は2つ＝冬の新人大会（第1）と翌夏の中体連（この代が3年として迎える集大成）。
- 月別の重心: 8月始動（点の取り方の土台＋DF基礎を3on3）→ 9月積み上げ → 10月アイソの磨き込み＋チームDF連携 → 11-12月 中野区新人大会 → 1-2月 都新人大会 → 3月移行 → 4月再編成（新1年加入）→ 5月仕上げ → 6-7月 中体連。年内（〜12月）の攻めはアイソレーションが上限でセットは組まない。
- 準備期の作り方: 孤立ドリルでなく、ハンドリング・近〜中フィニッシュ・DF基礎・ボックスアウトを1on1〜5on5の実戦に溶かし込む。準備と鍛錬の違いは人数でなく攻め方の質。
- 男女: 別チームだが体育館共有（火金=全面/一部共通・一部分割、水木=半面/ほぼ共有、変更可）。男女で発足時期がずれる（男子は都選手権まで進む見込みで後ろ倒し）。

## 開発中に得た教訓（次セッションのClaudeへ）

1. ペルソナ/スタイルが長い議論セッションで崩れる。何が起きたか=会話に乗って二人称「君/きみ」を多用（オーナーから2回叱責、正しくは「しんたろうさん」）、装飾の太字（**）を多用（やめろと指示）、「ファンダ期」等の辞書にない造語を使用（気持ち悪いと指摘）。再発防止=オーナーへの出力前に「しんたろうさん固定・太字なし・造語なし」を自己チェックする。反映先候補=CLAUDE.local.md（呼称は既出）／クロスセッション記憶。
2. ドキュメントの粒度を取り違えた。何が起きたか=チーム哲学（高レベル）に速攻の細かい運び方（誰が運ぶ・パス判断）を詰め込んだ。再発防止=哲学＝高レベルの identity、実装の機構＝戦術/ドリル層、と粒度を分ける。反映先候補=クロスセッション記憶。
3. 中学固有ルールを最初に確認すべきだった。何が起きたか=中学はゾーン禁止という前提を知らずにゾーンDFを提案して叱責。再発防止=競技・カテゴリ（中学/高校等）のルール前提を、設計提案の前にコーチへ確認する。反映先候補=クロスセッション記憶。
4. コーチング領域の前提を独断で置かない。何が起きたか=「大きい選手はドリブルさせない」「夏の大会がピーク」など旧態依然/誤った前提で書いて複数回差し戻し。再発防止=ドメイン判断は推測で固めず、コーチの哲学（ポジションレス等）を先に引き出してから設計に落とす。反映先候補=クロスセッション記憶。
